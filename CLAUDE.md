# CLAUDE.md

Hướng dẫn cho AI assistant khi làm việc với project này.

## Project Info

- **Tên project**: DevOps AI
- **Mục đích**: Học và thực hành DevOps từ cơ bản đến nâng cao
- **Ngôn ngữ giao tiếp**: Tiếng Việt

## Conventions

- Commit message bằng tiếng Anh, theo format: `type: description` (ví dụ: `feat: add login page`)
- Code comment bằng tiếng Anh
- Documentation (MD files) bằng tiếng Việt
- Sử dụng branching model: main / develop / feature/*

## Quy tắc

- Luôn đọc REQUIREMENTS.md trước khi implement tính năng mới
- Cập nhật TASKS.md khi bắt đầu hoặc hoàn thành task
- Cập nhật CHANGELOG.md khi release version mới
- Không commit file chứa secrets (.env, credentials)
